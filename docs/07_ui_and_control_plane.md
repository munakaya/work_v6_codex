# UI and Control Plane Overview

이 문서는 UI 관련 설계의 진입점이다. 실제 화면 설계와 프론트엔드 조회 계약은 아래 하위 문서로 분리되어 있다.

- `07_ui_design.md`: UI 범위, 화면 상세, 정보 구조, 핵심 사용자 흐름
- `07_frontend_query_contract.md`: 프론트엔드 조회 계약과 실시간 업데이트 정책

읽는 순서:

1. 어떤 화면을 만들지와 범위를 잡을 때 `07_ui_design.md`
2. 프론트엔드와 백엔드 조회 계약을 맞출 때 `07_frontend_query_contract.md`
