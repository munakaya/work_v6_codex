# Execution Plan Overview

이 문서는 실행 계획 관련 설계의 진입점이다. 진행 상태, 실행 계획, 리스크 추적은 아래 하위 문서로 분리되어 있다.

- `08_progress_and_gaps.md`: 현재 진행 상태, 남은 공백, 다음 우선 문서
- `08_delivery_plan.md`: 구현 묶음, 주차별 계획, 병렬 작업선, exit criteria
- `08_execution_risk_matrix.md`: 문제 우선순위, 계승할 강점, 리스크, 역추적 표

읽는 순서:

1. 지금 설계가 어디까지 왔는지 볼 때 `08_progress_and_gaps.md`
2. 실제 실행 순서와 팀별 분할을 잡을 때 `08_delivery_plan.md`
3. live 전환 전 리스크와 추적표를 확인할 때 `08_execution_risk_matrix.md`
